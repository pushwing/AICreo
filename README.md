# AICreo

> CodeIgniter 4 기반 웹에이전시 보일러플레이트 모음

## 디렉토리 구조

| 디렉토리 | 설명 |
|---|---|
| `default/` | 기본 홈페이지 템플릿 (기업 홈페이지, 소개 사이트) |
| `shop/` | 쇼핑몰 템플릿 (default 기반) |

## 공통 기능

- 동적 페이지 · 게시판 · 문의폼
- 회원 인증 (이메일 + 소셜 로그인 — Google · Naver · Kakao)
- 관리자 패널 (배너 · 팝업 · 메뉴 · 설정)
- 다중 테마 지원 (default / spring / dark / violet)
- SEO (OG태그 · GA · 네이버 웹마스터)

## 시작하기

각 디렉토리의 `SETUP.md`를 참고하세요.
